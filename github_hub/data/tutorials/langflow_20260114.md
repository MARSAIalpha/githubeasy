## 项目简介  
Langflow 是一款可视化 AI 工作流构建平台，无需编码即可设计、测试并部署基于大语言模型（LLM）的智能代理与自动化流程。

## 核心功能  
- **拖拽式可视化编辑器**：通过图形界面快速搭建 AI 流程，支持连接 LLM、向量数据库和工具组件。  
- **Python 自定义扩展**：所有组件均可用 Python 源码修改，满足高级定制需求。  
- **一键部署为 API/MCP 服务**：将流程直接发布为 REST API 或 MCP 工具，无缝集成到任何应用中。  
- **实时交互式调试**：内置沙盒环境，支持逐步骤运行与结果查看，加速开发迭代。  
- **企业级安全与可观测性**：支持 LangSmith、LangFuse 等监控工具，提供权限控制和部署扩展能力。

## 快速上手  
1. **安装依赖**（推荐使用 `uv`）：  
   ```bash
   uv pip install langflow -U
   ```

2. **启动服务**：  
   ```bash
   uv run langflow run
   ```

3. **打开浏览器访问**：  
   👉 http://127.0.0.1:7860  

> ✅ 首次运行会自动加载默认模板，无需额外配置。支持 Windows/macOS（推荐使用 [Langflow Desktop](https://www.langflow.org/desktop) 一键安装）。

## 使用示例  
创建一个简单的“问答机器人”流程：  
1. 在界面中拖入 **LLM** 组件（如 OpenAI GPT-4），填写 API Key。  
2. 拖入 **Text Input** 和 **Output** 组件，连接它们与 LLM。  
3. 在 LLM 的提示词模板中输入：`“请用一句话回答：{input}”`  
4. 点击运行，在右侧输入框输入：“什么是 Langflow？”  
5. 查看输出结果，即可看到 AI 生成的回答。

> 💡 可导出为 JSON 文件，或通过 API 调用该流程：  
> `curl -X POST http://localhost:7860/api/v1/run/your-flow-id -H "Content-Type: application/json" -d '{"input": "你好"}'`

## 常见问题  

**Q1：我应该使用 Langflow Desktop 还是命令行安装？**  
A：新手建议下载 [Langflow Desktop](https://www.langflow.org/desktop)，它内置 Python 环境，免配置即可使用。开发者或需要定制时再用 `pip install langflow`。

**Q2：为什么不能升级到 1.7.0 或 1.6.0 版本？**  
A：这些版本存在严重 Bug（如数据丢失、环境变量失效）和安全漏洞，请直接升级至 **1.7.1+** 或 **1.6.4+**，避免使用中间版本。

**Q3：如何将流程集成到我的 Python 应用中？**  
A：导出流程为 JSON 后，可通过 Langflow 提供的 SDK 或 HTTP API 调用。示例：  
```python
import requests
response = requests.post("http://localhost:7860/api/v1/run/your-flow-id", json={"input": "你好"})
print(response.json()["output"])
```
